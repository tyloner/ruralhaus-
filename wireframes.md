## Mobile Responsive Considerations

For mobile devices, the layout will adapt as follows:

1. **Navigation**: Convert to hamburger menu
2. **Multi-column layouts**: Stack vertically
3. **Property cards**: Display in single column
4. **Carousels**: Optimize for touch interaction
5. **Images**: Resize appropriately for smaller screens
6. **Font sizes**: Adjust for readability on mobile
7. **CTAs**: Ensure buttons are large enough for touch interaction

## Style Guide Elements

### Typography
- Headings: Serif font (e.g., Playfair Display) for rural aesthetic
- Body: Sans-serif font (e.g., Montserrat) for clean, modern feel
- Font sizes: Responsive scaling based on viewport

### Colors
- Primary: Earthy brown (#8B7355)
- Secondary: Forest green (#2E8B57)
- Accent: Ocean blue (#4682B4)
- Neutrals: Warm white (#F5F5F5), Light gray (#E0E0E0), Dark gray (#333333)

### UI Elements
- Buttons: Rounded corners with subtle hover effects
- Cards: Light shadows and subtle borders
- Icons: Simple, line-based designs
- Dividers: Thin lines or subtle texture patterns

### Images
- Property photos: High-quality, professionally shot
- Background images: Rural landscapes, natural textures
- Overlay text: Ensure readability with appropriate contrast
